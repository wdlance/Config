图片分组整理工具功能清单
一、核心基础功能

1. 到期限制
   设定固定到期时间（2026-05-20 00:00:00），到期后自动锁定所有功能
   锁定后效果：显示全屏锁定遮罩、页面元素置灰且禁用交互、按钮 / 输入框禁用、鼠标指针显示 “禁止” 样式
2. 文件导入
   支持两种导入方式：
   选择本地文件夹（通过 webkitdirectory 属性实现）
   选择 ZIP 压缩包（解析 ZIP 内目录结构和文件）
   导入过滤：仅保留图片 / 字体类文件（图片：jpg/jpeg/png/gif/bmp/webp；字体：ttf/woff/woff2/eot/otf）
   保留原文件目录结构，自动生成对应分组树
3. 分组管理（目录树）
   自动生成：导入文件时根据目录结构自动创建分组节点
   手动创建：在当前选中分组下新建自定义子分组（支持选中文件批量移入新分组）
   分组编辑：
   双击分组名称可重命名
   自定义分组支持删除（删除后文件归回根分组）
   分组节点显示文件数量（仅统计未删除文件）
   分组选中：点击分组节点切换当前操作分组，选中状态高亮显示
   层级展示：分组树按层级缩进显示，根分组无父级
   二、文件操作功能
4. 文件展示
   网格布局：可自定义每行显示列数（2-20 列），默认 6 列
   文件预览：
   图片文件双击可弹窗预览（ESC 关闭预览）
   字体文件显示通用字体图标占位
   文件信息：显示文件名（底部半透明遮罩，超出省略）
   排序：按文件名中的数字部分升序排列
5. 文件选择
   单点选择：点击文件切换选中状态（蓝色边框高亮）
   Shift 多选：按住 Shift 点击可选中连续范围文件
   框选：鼠标拖拽绘制选框，批量选中框内文件
   取消选中：点击 “取消选中” 按钮清空所有选中状态
6. 文件操作
   标记删除：点击文件右上角 “×” 按钮标记删除（仅标记，未真正删除）
   撤销删除 / 归回根分组：点击文件右上角 “↶” 按钮，将文件移回根分组
   拖拽排序：文件可在当前分组内拖拽调整顺序
   跨分组拖拽：选中文件拖拽到目标分组节点，批量移动文件
   三、交互操作
7. 快捷键支持
   ESC：关闭图片预览弹窗
   Shift：配合点击实现连续文件选择
8. 拖拽交互
   文件拖拽：支持文件在分组内 / 跨分组拖拽
   选框拖拽：空白区域拖拽生成选框，批量选择文件
   四、导出功能
9. 打包导出
   按左侧分组目录结构导出（不包裹外层根目录，保持原结构）
   自定义导出文件名（默认 “文件导出”）
   仅导出未标记删除的文件
   导出格式：ZIP 压缩包（依赖 JSZip 和 FileSaver 库）
   五、界面交互细节
10. 视觉反馈
    选中文件：蓝色粗边框高亮
    选中分组：浅蓝色背景高亮
    自定义分组：名称标蓝 +“可编辑” 提示
    操作按钮 hover 效果：轻微透明度变化
    预览弹窗：黑色半透明背景，图片自适应显示
11. 布局适配
    响应式容器：分组树宽度固定 320px，图片面板占剩余宽度
    滚动处理：分组树 / 图片面板独立滚动，框选功能兼容滚动位置
    自适应网格：图片网格列数可实时调整，自动适配布局
    六、技术依赖
    第三方库：JSZip（ZIP 解析 / 生成）、FileSaver（文件下载）
    原生 API：File/Blob/URL.createObjectURL（文件预览）、DragEvent（拖拽）、getBoundingClientRect（框选）
    七、核心规则
    目录结构：导出时严格保留分组层级，根分组不生成外层文件夹
    文件状态：标记删除的文件不参与显示、计数、导出
    交互优先级：到期锁定功能优先于所有其他操作
    数据存储：所有数据仅在前端内存中处理，无本地存储 / 后端交互
